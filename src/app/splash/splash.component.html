Welcome to ngTetris

I've decided to move from AngularJS to the new Angular.io, so why not a Tetris clone as first project instead of boring Hello Worlds? Here I'm basically experimenting a bit with components and services, and have some fun with a lot of left floating DIVs.

The rules should be pretty similar to the original game, the more lines you clear in one move, the more points you get. Clearing 4 lines is "Tetris", and each time you do, it awards more points.

Also I think this is the first time that I program Tetris as I always considered it a boring game, but also I never really played it until now, which is a shame.

Anwyay, I'd like to thank gabrielwoj for the sounds effects, they're awesome :) I couldn't find the music though, and so had some fun with FL Studio and created my own. It's not that great, but I like how it sounds.
